# CCAPDEV

To run react app:

1.  Navigate to ROOT FOLDER (CCAPDEV folder) using console/terminal within vs code
    1.1 Shortcut for windows is ctrl+shift+`
    1.2 to go to folder command is

             cd "folder name"

2.  To run web application use command

        npm start
